# Cucumber-BDD
FrontEnd using Selenium-Java and RestAssured API BDD Automation.

Download or Clone the solution from Git Repository.
Go to the project directory in command prompt.
Run the tests with using command - mvn test.
You can see the result in HTML Report and also json and xml reports are available.
This solution includes both failure and success scenarios, the failed Test case is a negative scenario which does not match the expected result.


Pre-requisite:-
Install Java.
Install Maven.
